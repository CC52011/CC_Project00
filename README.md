# CC_Project00
This repository collects practical browser extension scripts for public use, aiming to bring convenience to users.
本仓库收录多款实用浏览器扩展脚本，开源免费，助力日常效率提升。

## 使用教程｜Installation Guide
### 中文安装步骤
1. 下载项目源码压缩包并解压
2. Edge浏览器地址栏输入：`edge://extensions/`
3. 在扩展管理页面开启「开发人员模式」
4. 点击「加载解压的扩展程序」
5. 选中已经解压完成的项目文件夹
6. 打开BOSS直聘网页端，手动筛选求职城市、目标岗位
7. 点击浏览器右上角扩展图标，先点击【扫描】，再点击【开始】执行自动投递

### English Installation Steps
1. Download and unzip the source code package
2. Enter `edge://extensions/` in Edge address bar
3. Turn on Developer Mode on the extension page
4. Click the option "Load unpacked"
5. Select your unzipped project folder
6. Log into BOSS Zhipin website and filter target city & job categories manually
7. Open the extension icon, tap `Scan` first, then click `Start` for auto-resume delivery

## 插件说明｜Introduction
一款面向求职者的效率工具，依托浏览器扩展实现BOSS直聘岗位简历批量自动投递。
An efficiency tool for job seekers, which realizes bulk automatic resume submission on BOSS Zhipin via browser extension.
