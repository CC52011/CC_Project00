# BOSS 海投助手 v2

v2 是重写版，专门适配 BOSS 搜索页常见的“左侧岗位列表 + 右侧岗位详情”布局。

它不再跳转岗位详情页，也不再读取公司链接。当前流程是：

1. 扫描左侧岗位卡片。
2. 点击左侧岗位卡片上方的岗位标题区域。
3. 等右侧岗位详情刷新。
4. 点击右侧 `立即沟通`、`与BOSS随时沟通` 等按钮。
5. 在弹窗里点击 `继续沟通`。
6. 进入聊天页后填写“给 HR 的留言”并点击 `发送`。
7. 返回上一页。
8. 点击左侧下一个岗位，重复流程。
9. 当前可见列表到底后，滚动左侧列表/鼠标滚轮加载更多岗位。

## 安装

1. 打开 Edge：`edge://extensions/`
2. 打开“开发人员模式”。
3. 点击“加载解压缩的扩展”。

## 使用

1. 登录 BOSS。
2. 在搜索页筛好城市、岗位方向、经验、学历等条件。
3. 点击扩展图标，打开 `BOSS 海投助手 v2`。
4. 先把“单次最多处理”设成 `3`。
5. 填写 HR 留言。
6. 点击“扫描”，确认发现岗位数大于 0。
7. 点击“开始”。
8. 观察右下角 v2 状态面板和页面反应。

## 日志状态

- `continue`：已点击右侧沟通按钮和弹窗里的继续沟通，等待进入聊天页。
- `sent`：进入聊天页后填写了留言，并点击了发送。
- `clicked`：点到了沟通/投递按钮，但聊天页没有找到可填写输入框或发送按钮。
- `skip`：被黑名单、同公司规则、已处理记录或按钮缺失跳过。
- `error`：处理时出现异常。

如果一直是 `clicked` 而不是 `sent`，通常说明聊天页输入框或发送按钮没有被页面开放，或者需要你手动完成验证。扩展不会绕过登录、验证码、安全验证或 App 限制。

## 建议设置

- 单次最多处理：先 `3`，确认正常后再改 `50` 或 `100`
- 最小间隔秒：`4`
- 最大间隔秒：`9`
- 自动翻页：开启后会在当前可见岗位处理完后滚动加载更多
- 跳过已发送/已点击记录：开启
- 同公司只处理一次：重复公司很多时开启

## 和旧版区别

- v2 不跳转详情页。
- v2 不使用公司链接。
- v2 失败记录不会永久阻止重试。
- v2 会跨页面记住当前岗位，聊天页发送后自动返回搜索页继续。
